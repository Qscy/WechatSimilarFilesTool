# WechatSimilarFilesTool 微信相似文件查找工具
## 为什么要做这个
* ~~工作需要，微信不开启自动下载则会错过很多文件，到头来还得找人重发~~
* ~~为了避免社恐人需要经常经历宇宙大爆炸，必须开启自动下载~~
* ~~但是微信如果开启自动下载会重复存储很多一样的文件~~
* ~~所以需要删除重复的文件，优化储存空间~~
* 好玩
